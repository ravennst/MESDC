# MESDC
MES Dynamic Configuration (MESDC) - Dynamically reads all active MES (Modular Encounters Systems) faction configurations after mod load and generates a human-readable baseline reference file.
Allows server admins to override MES spawn behavior via a single editable config file without requiring per-faction override mods. Adds per-faction region locking and exclusion zones.

Server-side mod. Clients download for definition registration only. Requires Modular Encounters Systems.
